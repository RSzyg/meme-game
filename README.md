# meme-game
A series of games about meme.  
Also a reconstruction of fighting-meme by using html canvas.  
## key description
### player1
`←` is move left  
`↑` is up(or jump)  
`→` is move right  
`/` is attack  
### player2
`D` is move left  
`R` is up(or jump)  
`G` is move right  
``` is attack
