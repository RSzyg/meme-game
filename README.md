# meme-game
A reconstruction of fighting-meme by using html canvas.  
~~Also a series of games about meme.~~  
## key description
### player1
`←` is move left  
`↑` is up(or jump)  
`→` is move right  
`/` is remote attack  
`.` is defense  
### player2
`D` is move left  
`R` is up(or jump)  
`G` is move right  
`1` is attack  
`` ` `` is defense
